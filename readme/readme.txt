Plugin for CudaText.

About Codeium
A Free AI-Powered Toolkit for Developers. https://codeium.com/

About addon
AI Autocomplete using Codeium (free Copilot alternative)

Installation
Run command Codeium: Get new token... Web page will be opened.
Log in (or register) and skip to the page with a token.
Copy token to clipboard, paste it to CudaText dialog which asks for it.
Now you should be logged in. Server executable will be downloaded and started automatically.

Do not forget to set hotkey for Codeium: Get completions command. For example Alt+\.

Enjoy!
